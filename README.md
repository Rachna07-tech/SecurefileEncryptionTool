# Secure File Encryption Tool

## Overview

The Secure File Encryption Tool is a Python desktop application that protects files using the Fernet implementation from the `cryptography` library.

## Features

- Generate encryption key
- Encrypt files
- Decrypt files
- Browse files using a GUI
- Error handling
- Supports multiple file types

## Technologies

- Python
- Tkinter
- Cryptography (Fernet)
- VS Code

## How to Run

1. Activate the virtual environment.
2. Run:

python main.py

3. Generate a key.
4. Browse a file.
5. Encrypt or decrypt the file.
